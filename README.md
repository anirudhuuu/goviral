# GoViral

A live streaming simulation platform for practicing and recording video content with real-time analytics and AI-generated viewer interactions.

## Features

- Live video streaming with camera and microphone support
- Vertical and horizontal streaming orientations
- Real-time speech recognition and analysis
- Practice mode with performance metrics (WPM, filler words, confidence score)
- AI-generated viewer comments and reactions
- Video recording with downloadable MP4/WebM format
- Interactive chat with emoji support
- Device selection for camera and microphone
- Quality settings (360p, 720p, 1080p)
- Video filters and effects

## Usage

1. Navigate to the setup page
2. Enter your name and stream topic
3. Select streaming orientation (vertical or horizontal)
4. Configure camera, microphone, and quality settings
5. Click "Go Live" to start streaming
6. Use practice mode to view real-time performance metrics
7. End stream to view summary and download recording

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Radix UI
- Framer Motion
- Google Gemini API
